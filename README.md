## Lemonade App (Jetpack Compose)

## Overview

The Lemonade App is a simple interactive Android application built using **Kotlin** and **Jetpack Compose**.
It simulates the process of making lemonade through a sequence of user interactions.

This project demonstrates **state management, UI recomposition, and user interaction handling** in a modern Android development setup.

---

## Features

* State-driven UI using Jetpack Compose
* Interactive click-based transitions
* Randomized squeeze logic for realistic behavior
* Accessibility support using content descriptions
* Clean and modular composable structure

---

## Concepts Used

* Jetpack Compose UI
* State management (`mutableIntStateOf`)
* Conditional rendering (`when` expressions)
* Event handling (`Modifier.clickable`)
* Accessibility (TalkBack support)

---

## App Flow

1. Tap the lemon tree to pick a lemon
2. Tap multiple times to squeeze the lemon
3. Drink the lemonade
4. Restart the process

---

## Tech Stack

* Kotlin
* Jetpack Compose
